Projects

AcadPro - Student Management System

AcadPro is a full-stack web application designed to streamline student data management for educational institutions. Built using Node.js, Express.js, MongoDB, and Mongoose, it features a responsive frontend with HTML, CSS, and JavaScript. The system supports role-based access for Admins, Teachers, and Students, with modules for managing student records, attendance, courses, and grades. Integrated tools like Winston, Morgan, Dotenv, and CORS enhance logging, security, and performance. AcadPro significantly improves operational efficiency and user experience while reducing system load and bounce rates.

Brew & Bloom - A Fully Functional Coffee Shop Website with Stripes Integration!

A fully functional e-commerce-style coffee shop website featuring real-time cart updates, quantity management, and secure Stripe payment integration. Built using Node.js, Express.js, HTML, CSS, JavaScript, and Stripe API, with LocalStorage for cart persistence and dotenv for secure configuration. Designed for performance and user experience — fast, responsive, and checkout-ready.Stripe Integration enabled real-time, secure transactions, streamlining the payment process and enhancing trust.LocalStorage & optimized DOM handling reduced cart data loss and improved performance by 90% Improved load time and checkout flow resulted in an estimated 70% decrease in bounce rate.
